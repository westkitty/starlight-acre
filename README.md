# Starlight Acre Startup Asset Pack

Welcome to the Starlight Acre bootstrap project! This directory contains all the visual foundation needed for a mythic orbital farming station. 

## Quick Start
1.  **Assets**: Located in `/assets/`.
2.  **Guide**: See `/assets/docs/style_guide.md` for the color palette and visual rules.
3.  **Manifest**: See `/assets/docs/asset_manifest.md` for a full file list and slicing dimensions.
4.  **Usage**: See `/assets/docs/asset_usage_notes.md` for Godot 4.x integration tips.

## Folder Overview
- `branding/`: Logo, emblem, title screen (Vector & PNG).
- `sprites/`: Player animations and Wisdom Fruit crop states.
- `ui/`: HUD icons and interaction prompts.
- `backgrounds/`: Greenhouse environment.
- `tilesets/`: Modular floor/wall panels.
- `effects/`: Pixel art VFX.
- `docs/`: All project documentation.

## Tone & Vibe
- **Style**: 16-bit / 32-bit Pixel Art (GBA/SNES).
- **Mood**: Mythic Sci-Fi, Restrained Wonder.
