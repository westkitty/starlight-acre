# Starlight Acre Style Guide

## Aesthetic Pillar: GBA/SNES Pixel Art
The visual direction is a nostalgic yet clean 16-bit to 32-bit era handheld aesthetic. 

## Palette Reference
- **Deep Space Navy**: #0A0E1A (Base background, deep shadows)
- **Dusk Blue**: #1E2D4A (Mid-tones, tech panels)
- **Teal**: #2E8B8B (Hydroponics, nutrient glow)
- **Moss Green**: #4F7942 (Plant life, healthy growth)
- **Warm Gold**: #D4AF37 (Wisdom Fruit, mythic accents)
- **Copper**: #B87333 (Maintenance hardware, pipes)
- **Stone Gray**: #708090 (Station structure, floors)
- **Bone/Off-white**: #F5F5DC (Text, bright highlights)

## Design Rules
- **Pixel Consistency**: All gameplay assets should maintain a consistent pixel scale.
- **Outlines**: Use dark, subtle outlines for characters and interactables to ensure readability against backgrounds.
- **Grit vs. Wonder**: Hardware should look used (copper, rust) while crops should look ethereal and bioluminescent (gold, teal).
- **UI Clarity**: High contrast icons for resources. Use the "E" icon for all interaction prompts.

## Suggested Fonts
- **Silver** (by Poppy Works): Accessible pixel font.
- **m5x7** (by Daniel Linssen): Compact and highly readable.
